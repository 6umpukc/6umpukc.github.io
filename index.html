шаги УСТАНОВКИ решения

Как запустить САЙТ ЛОКАЛЬНО И протестировать РешениЕ на базе продуктов «1С-Битрикс»

шаг 1

УСТАНОВИТЕ LOCAL

Local это простой способ запускать или разрабатывать сайты на WordPress, Bitrix и других платформах локально.

Посмотреть видео процесса установки >>>

шаг 2

СОЗДАйТЕ САЙТ ИЗ ПОДГОТОВЛЕННЫХ НАСТРОЕК BITRIX

Скачайте нужную версию настроек для сайта Bitrix и запустите bitrixsetup.php для его установки. Быстрая установка позволяет в упрощенном виде установить продукт «1C-Битрикс».

Посмотреть видео процесса создания сайта >>>

шаг 3

УСТАНовите решение ИЗ Маркетплейс

После завершения установки Bitrix выберите решение для установки в мастере или в административном разделе.

Посмотреть видео процесса установки решения >>>

Готовые настройки для сайта на Bitrix

UTF-8

для Bitrix в кодировке UTF-8

WINDOWS-1251

для Bitrix в кодировке WINDOWS-1251

UTF-8 LEGACY

для старых версий Bitrix в кодировке UTF-8

BLANK-SITE

для реализации приложения Bitrix24 или простого сайта

UTF-8 PHP5.6

для очень старых версий Bitrix в кодировке UTF-8
